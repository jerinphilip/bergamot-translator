class Response¶

Response holds AnnotatedBlob(s) of source-text and translated text, alignment information between source and target sub-words and sentences.

AnnotatedBlob provides an API to access markings of (sub)-word and sentences boundaries, which are required to interpret Quality and Alignment (s) at the moment.

const size_t size() const¶

Number of sentences translated.

The processing of a blob of into sentences are handled internally, and this information can be used to iterate through meaningful units of translation for which alignment and quality information are available.

Access to histories, which holds rich information on translated text.

Not recommended to use, will be removed in future.

-logprob of each word and negative log likelihood of sequence (sentence) normalized by length, for each sentence processed by the translator.

Each Alignment is a sparse matrix representation with indices corresponding to (sub-)words accessible through Annotation.
