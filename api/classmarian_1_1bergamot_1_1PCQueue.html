template<class T>
class PCQueue¶

Producer consumer queue safe for multiple producers and multiple consumers.

T must be default constructable and have operator=. The value is copied twice for Consume(T &out) or three times for Consume(), so larger objects should be passed via pointer. Strong exception guarantee if operator= throws. Undefined if semaphores throw.
