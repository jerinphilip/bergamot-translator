class Service¶

Service exposes methods to translate an incoming blob of text to the Consumer of bergamot API.

An example use of this API looks as follows:

options = …; service = Service(options); std::string input_blob = “Hello World”; std::future<Response> response = service.translate(std::move(input_blob)); response.wait(); Response result = response.get();

Optionally Service can be initialized by also passing model_memory for purposes of efficiency (which defaults to nullpointer and then reads from file supplied through config).
